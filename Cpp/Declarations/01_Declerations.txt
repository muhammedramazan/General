<---->  : Declarations are how names are introduced (or re-introduced) into the C++ program. 
        : Not all declarations actually declare anything, and each kind of entity is declared differently.
        : Definitions are declarations that are sufficient to use the entity identified by the name.

- A declaration is one of the following:
[ ]  -> Function definition
[ ]  -> Template declaration (including Partial template specialization)
[ ]  -> Explicit template instantiation
[ ]  -> Explicit template specialization
[ ]  -> Namespace definition
[ ]  -> Linkage specification
[ ]  -> Attribute declaration (attr ;) (since C++11)
[ ]  -> Empty declaration (;)
[ ]  -> A function declaration without a decl-specifier-seq:
